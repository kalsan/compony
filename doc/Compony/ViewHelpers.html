Instance Method Details

#render_intent(button: {}) ⇒ Object

Renders a button/link to a component given a comp and model or family. If authentication is configured and the current user has insufficient permissions to access the target object, the link is not displayed. When inside a request context (content do...), this is preceded by RequestContext#render_intent.

Parameters:

  • button (Hash) (defaults to: {}) —

    Parameters that will be given to the button component initializer.
